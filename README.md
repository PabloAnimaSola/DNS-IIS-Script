# DNS-IIS-Script
Para correrlo, recuerda tener bind, ufw, y apache ya instalado.
Para clonarlo:\
``git clone https://github.com/PabloAnimaSola/DNS-IIS-Script.git``

¡Para ejecutarlo usa sudo, o saldrá error!\
``cd ~/DNS-IIS-Script``\
``sudo ./creadorDNS-IIS.sh``\
o\
``sudo bash ~/DNS-IIS-Script/creadorDNS-IIS.sh``\
Si sigue sin ejecutar añade permisos de ejecución:\
``sudo chmod +x ~/DNS-IIS-Script/creadorDNS-IIS.sh``
